BeaverOps Agent
=====================================

Run this on the same PC as Timberborn.
It bridges your local game to the BeaverOps relay so you can
view the dashboard from beaverops.com on any device.

HOW TO USE:
  Windows: Double-click start-agent.bat
  Mac/Linux: python3 agent.py

REQUIREMENTS:
  Python 3 — https://www.python.org/downloads/
  Internet connection

The agent will show you an 8-character session code like:
  BVRS-4829

Enter that code at beaverops.com to connect.
